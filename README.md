# MotoConnect

MotoConnect es una plataforma diseñada para centralizar la búsqueda de repuestos y accesorios para motociclistas. Actualmente, se encuentra implementada como una API desarrollada en NestJS, con un frontend en React que permite interactuar con las funcionalidades principales del sistema.
El objetivo a corto plazo es ofrecer una experiencia fluida para los usuarios que deseen registrarse, iniciar sesión y buscar productos relacionados con el mundo motero. A mediano plazo, se proyecta integrar un sistema de pagos y compras online, evolucionando hacia un marketplace completo de accesorios y repuestos para motos.
